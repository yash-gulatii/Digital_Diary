About Project

This Project is about a "Digital Diary" where you can keep record of your emotions, feeling, ideas, creativity, daily life and much more in the form of text.

The Backend of this project is Python with Django Framework, for Frontend we have used HTML & CSS and for Database we have used SQLite.
